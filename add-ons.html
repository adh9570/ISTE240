Additional Expansions


There are a lot of Linux add-ons to choose from. The first is an office suite. Your Linux distribution might actually already come with a free, pre-installed office suite. If not you can always download one like Libreoffice, which has all the features of a word editor. If you don’t like this, you can always use Google Docs. Another add-on to look into is a web browser. No doubt you already have your favorite or preferred browser, but you are experimenting with Linux, so why not try a new browser like Chromium, or Midori? Another add-on that Linux brings to the table is your email viewer. Try out a different program and finds one that best fits you such as Thunderbird, Evolution, or KMail. There are so many add-ons available for Linux that we just can’t possibly list them all here! Check out the most popular add-ons here.


Below is a descrption of a few add-ons that are currently the most popular for Linux users:


  • VLC Media Player


    • The VLC media player is exactly what it sounds like: a media player. It can do so much more than just play video files you find and download, though. It actively finds and downloads codecs for unknown file formats and allows you to play them. This elimiates future playback errors. This software also allows you to play a DVD or stream the video to another device.

      Click here to download.


  • Mozilla Firefox


    • Firefox is the pre-installed web browser in most Linux distributions. It is popular because of is fluid interface. You are able to play YouTube videos right away, and can download extensions to do the same with other video formats. It also updates on launch to ensure you are always using the most current version available. As always, if you don't like Firefox you can always download and install any browser you like.

      Click here to download.


  • GIMP (GNU Image Manipulation Program)


    • GIMP is free image editor used to re-touch and customize images. You might need some time to adjust to the interface if you like a different editor, but GIMP is well worth it. It can do everything a high-end editor can do and supports photoshop plug-ins. An added benefit is that the software is less than 100MB! Great for conserving disk space.

      Click here to download.


  • Thunderbird E-mail Client


    • Thunderbird is a great, free, customizable e-mail client. It has a set-up wizard that guides you through the process to set-up your exsisting e-mail, or create a new one. You can add as many accounts as you want! Thunderbird is also 100% customizable. You can change the theme, install add-ons, and change the folder layout and sorting system.

      Click here to download.


  • Libreoffice


    • Libreoffice is a full-blown office suite that is the rival to Microsoft Office. This office suite can do pretty much everthing Microsoft Office can. You can create and edit word files, powerpoints, excels, and math files. Libreoffice can even open Microsoft Office files as well.

      Click here to download.
